# Layerbounce AddOn

Layerbounce is a Classic Era addon that helps others switch between different layers.

## Features

- **Assisted Layer Switching** - Helps others swap layers

## Installation

1. Download the latest version of Layerbounce.
2. Extract the contents to your World of Warcraft Classic Era AddOns directory:
    ```
    C:\Program Files (x86)\World of Warcraft\_classic_era_\Interface\AddOns\Layerbounce
    ```
3. Restart World of Warcraft.

## Usage

1. Open World of Warcraft Classic Era.
2. Ensure the Layerbounce addon is enabled in the AddOns menu.
3. Use the minimap button to toggle the addon on and off.
